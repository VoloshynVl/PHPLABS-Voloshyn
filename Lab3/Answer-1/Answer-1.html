Звіт: Потік виконання, умовні конструкції та цикли у PHP

1. Потік виконання у PHP

  • Потік виконання — це послідовність команд, що виконує програма зверху вниз.
  • Змінюється за допомогою умов (if) та циклів (for, while).
  • Наприклад, блок if може змінити хід виконання залежно від умови.

2. Умовні конструкції

  • if: if ($a > 5) { ... }
  • if-else: if (...) { ... } else { ... }
  • if-elseif-else: для кількох умов
  • Вкладені умови: if (...) { if (...) { ... } }
  • Тернарний оператор: $result = ($a > 0) ? "yes" : "no";
  • switch: використовується, коли багато варіантів: 
    switch ($color) {
    case "red": echo "Червоний"; break;
    case "blue": echo "Синій"; break;
    default: echo "Інший";
}

3. Основні цикли у PHP

  • for: цикл з лічильником
    for ($i = 0; $i < 5; $i++) { echo $i; }
  • while: виконується поки умова істинна
    while ($a < 10) { $a++; }
  • do-while: мінімум одна ітерація
    do { ... } while ($a < 5);
  • foreach: для масивів
    foreach ($arr as $item) { echo $item; }

4. Управління потоком в циклах

  • break: вихід із циклу достроково
    if ($x == 5) break;
  • continue: перехід до наступної ітерації
    if ($x % 2 == 0) continue;
  • goto: переходить до мітки в коді
    goto end; ... end: echo "Кінець";
  • Обмеження goto: не можна переходити в/з функцій, циклів, або класів.
  • Нескінченний цикл: while(true) { ... }
  • Завершується вручну — через break або умову в середині тіла циклу.
