Звіт: Потік виконання, умови та цикли в PHP

1. Потік виконання в PHP

  • Потік виконання — це порядок, у якому PHP виконує команди.
  • За замовчуванням код виконується зверху вниз.
  • Умови (if, switch) та цикли (for, while) змінюють хід виконання.

2. Відмінності між if, if-else, if-elseif-else

  • if: перевіряє одну умову.
  • if-else: одна умова — два варіанти дій.
  • if-elseif-else: багато умов — обирається одна, що перша виконується.
  • Приклад: 
    
if ($a > 0) {
    echo "Позитивне";
} elseif ($a < 0) {
    echo "Негативне";
} else {
    echo "Нуль";
}

3. Тернарний оператор

  • Скорочена форма if-else.
  • Синтаксис: умова ? значення_якщо_так : значення_якщо_ні;
  • Приклад: $msg = ($age >= 18) ? "Дорослий" : "Неповнолітній";

4. Основні цикли в PHP

  • for: відома кількість повторів.
    for ($i = 0; $i < 5; $i++) { echo $i; }
  • while: виконується, поки істинна умова.
    while ($a < 10) { $a++; }
  • do-while: мінімум одна ітерація.
    do { ... } while ($a < 5);
  • foreach: перебирає масиви.
    foreach ($arr as $item) { echo $item; }

5. Оператори break, continue, goto

  • break: припиняє виконання циклу.
    if ($x == 5) break;
  • continue: переходить до наступної ітерації.
    if ($x % 2 == 0) continue;
  • goto: переходить до заданої мітки в коді.
    goto end; ... end: echo "Кінець";
  • Використовується рідко через зниження читабельності.
