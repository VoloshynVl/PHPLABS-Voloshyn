Звіт: Робота з XAMPP та PHP

1. Що таке XAMPP

  • XAMPP — це локальний сервер для розробки.
  • Складається з Apache, MySQL/MariaDB, PHP, phpMyAdmin.

2. Встановлення та налаштування XAMPP

  • Завантажити XAMPP із офіційного сайту.
  • Встановити програму та відкрити Control Panel.
  • Увімкнути Apache та MySQL.
  • PHP-файли зберігати в папці htdocs.
  • Перевірити роботу через http://localhost/.

3. Налаштування IDE для PHP

  • Вказати шлях до PHP-інтерпретатора.
  • Налаштувати сервер та шлях до проєкту.
  • Встановити URL запуску (наприклад, http://localhost/myproject).

4. Переваги IDE (PhpStorm, VS Code)

  • Автодоповнення коду.
  • Зручна відладка.
  • Робота з Git.
  • Підсвітка синтаксису.
  • Інтеграція з базами даних.

5. Перевірка роботи XAMPP

  • Запустити Apache в Control Panel.
  • Відкрити браузер і перейти на http://localhost/.
  • Якщо відкрилась стартова сторінка XAMPP — все працює.
