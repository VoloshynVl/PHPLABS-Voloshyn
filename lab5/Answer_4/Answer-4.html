Звіт: Робота з формами у PHP

1. Метод передачі даних у URL

  • Метод GET передає параметри у URL-рядку.
  • Наприклад: form.php?name=Ivan&age=20

2. Отримання даних з форми (POST)

  • Для доступу до значень використовується масив $_POST.
  • Приклад: $name = $_POST['name'];

3. Що таке валідація даних

  • Валідація — це перевірка правильності та безпечності введених користувачем даних.
  • Наприклад, перевірка, що поле email дійсно є електронною адресою.

4. Видалення пробілів у рядку

  • Використовується функція trim().
  • Приклад: $clean = trim($input);

5. Ризики без валідації

  • Можливість SQL-ін'єкцій або XSS-атак.
  • Для зменшення ризиків:
    • Очищення та перевірка даних.
    • Використання функцій: htmlspecialchars(), filter_input().
    • Підготовлені запити при роботі з базою даних.
