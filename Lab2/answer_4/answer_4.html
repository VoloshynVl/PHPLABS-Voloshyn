Звіт: Основи PHP

1. Що таке змінна у PHP?

  • Змінна — це контейнер для зберігання даних.
  • Оголошується знаком $ перед іменем (наприклад, $name).
  • Правила іменування:
    • Ім’я починається з $ та літери або підкреслення.
    • Може містити цифри, але не починатися з них.
    • Регістрозалежна (тобто $var ≠ $Var).

2. Основні типи даних у PHP

  • integer (ціле число)
  • float (число з плаваючою точкою)
  • string (рядок)
  • boolean (true/false)
  • array (масив)
  • object (об'єкт)
  • NULL
  • Різниця:
    • integer: ціле число, напр. 10
    • float: десяткове число, напр. 10.5

3. Перетворення типів у PHP

  • Неявне: PHP сам змінює тип за контекстом.
    $sum = "10" + 5; // результат: 15
  • Явне: використовується приведення типу.
    $num = (int)"25.7"; // результат: 25

4. Структура простого PHP-скрипта

  • Скрипт починається з <?php і закінчується ?>.
  • Можна використовувати функції, напр. sqrt(), pow().
  • Приклад: 
    <?php
$number = 9;
echo sqrt($number); // виведе 3
?>

5. Конкатенація у PHP

  • Об'єднання рядків виконується оператором ..
  • Приклад: $fullName = $firstName . " " . $lastName;
  • Результат: якщо $firstName = "Ivan", $lastName = "Petrenko", то отримаємо Ivan Petrenko.
