Звіт: Масиви та функції в PHP

1. Типи масивів у PHP

  • Індексовані масиви: ключі — числа
    $arr = [10, 20, 30];
  • Асоціативні масиви: ключі — рядки
    $user = ["name" => "Ivan", "age" => 25];
  • Багатовимірні масиви: масиви всередині масивів
    $matrix = [[1,2],[3,4]];

2. Основні операції з масивами

  • Створення: $a = array(1, 2, 3);
  • Додавання елементів: $a[] = 4;
  • Читання значення: $a[0];
  • Зміна: $a[1] = 99;
  • Видалення: unset($a[2]);

3. Вбудовані функції для масивів

  • count($arr) — кількість елементів
  • array_push($arr, $val) — додати в кінець
  • array_pop($arr) — видалити з кінця
  • array_merge($a, $b) — об'єднання масивів
  • in_array($val, $arr) — перевірка наявності елемента
  • array_keys($arr), array_values($arr)

4. Створення функцій у PHP

  • Оголошення:
    function greet($name) { return "Hello, $name"; }
  • Виклик: greet("Ivan");
  • Параметри можуть бути обов'язковими або з умовчанням
  • Функції можуть повертати значення через return

5. Оптимізація масивів і функцій

  • Використовувати foreach замість for для читабельності
  • Не копіювати великі масиви — передавати за посиланням: function test(&$arr)
  • Уникати повторного перебору одного й того ж масиву
  • Виносити повторюваний код у функції
  • Використовувати корисні функції: array_map(), array_filter()
