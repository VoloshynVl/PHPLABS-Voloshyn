Звіт: Масиви та функції в PHP

1. Які типи масивів підтримуються у PHP?

  • Індексовані масиви — ключі числові (0, 1, 2...).
  • Асоціативні масиви — ключі рядкові.
  • Багатовимірні масиви — масиви, які містять інші масиви.

2. Функція для сортування масиву за значеннями у зворотному порядку

  • Використовується функція rsort($array);
  • Вона змінює початковий масив, сортує його за значеннями від більшого до меншого.

3. Що повертає функція array_keys()?

  • Повертає масив, який містить усі ключі переданого масиву.
  • Приклад: array_keys(["name" => "Ivan", "age" => 25]) → ["name", "age"]

4. Синтаксис оголошення користувацької функції у PHP

  • function ім'я_функції(аргументи) { тіло функції }
  • Приклад:
    function greet($name) { return "Привіт, $name"; }

5. Як уникнути копіювання великого масиву при передачі у функцію

  • Передавати масив за посиланням, використовуючи символ &:
  • function processBigArray(&$data) { ... }
  • Це дозволяє функції працювати з оригінальним масивом без створення копії.
